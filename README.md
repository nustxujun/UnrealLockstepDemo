# UnrealLockstepDemo
A lockstep demo with deterministic 3d physics for UE.
# Features
- [x] lockstep framework on c++
- [x] deterministic 3d physics
- [x] serialization
- [x] snapshot
- [x] verification on server
- [x] rollback
- [x] replay tools in editor
- [ ] prediction
- [ ] lua supported
- [ ] async
